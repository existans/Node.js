# Репозиторій для вивчення предмету "Скриптові мови програмування"

Цей репозиторій призначений для виконання лабораторних робіт з дисципліни  
**"Скриптові мови програмування"** в Чернігівській політехніці на 4 курсі.

В ньому зберігаються всі лабораторні роботи та код, виконаний в рамках вивчення
основних скриптових мов, таких як **Node.js**, **JavaScript** та інших технологій,
що використовуються для серверного програмування.

## Структура репозиторію

- **lab1** — Папка з файлами першої лабораторної роботи  
- **lab2** — Папка з файлами другої лабораторної роботи  
- **lab3** — Папка з файлами третьої лабораторної роботи  
- **lab4** — Папка з файлами четвертої лабораторної роботи  
- **lab5** — Папка з файлами п’ятої лабораторної роботи  
- **Reports** — Папка з усіма готовими звітами лабораторних робіт  
- **README.md** — Опис репозиторію  
- **.gitignore** — Файл для ігнорування непотрібних файлів
